## Project Description

This project implements:

### Question 1
A file encryption and decryption system using custom shift rules.  
The program reads input from a file, encrypts it, decrypts it, and verifies correctness.

### Question 2
An arithmetic expression evaluator using:
- tokenization
- recursive descent parsing
- expression tree generation
- evaluation with correct precedence

The output strictly matches the required format.
